# IdeaVault 

**IdeaVault** is a dynamic web-based platform designed for dreamers, innovators, and entrepreneurs. It allows users to share groundbreaking startup ideas, explore concepts posted by the community, and refine those ideas collectively through interactive discussions and feedback validation.

---

## 🚀 Key Features

*   🔐 **Secure Firebase Authentication & JWT Security:** Real-time email/password and Google login integration backed by JSON Web Tokens (JWT) to secure user interactions, maintain sessions flawlessly on browser reload, and protect private routes.
*   🔍 **Advanced Search & Multi-Criteria Filtering:** An ultra-responsive exploration system allowing users to search startup ideas instantly by title using case-insensitive `$regex`, alongside a categorical filtering system.
*   💬 **Dynamic Interactive Comment System (Full CRUD):** Logged-in users can engage with ideas by adding, editing, or deleting their comments in real-time. Comments feature live user badges, timestamps, and deep-nested security.
*   📊 **Personalized Dashboards (My Ideas & Interactions):** Comprehensive private panels for users to manage their posted ideas via clean update/delete modals, plus a custom interaction tracker displaying every concept they've actively reviewed.
*   🌓 **Global Light/Dark Theme & Flawless UX:** A fully responsive, pixel-perfect layout using modern UI structures, consistent spacing, interactive toast alerts (no default alerts used), and a global theme toggler built into the navbar.

---

## 🛠️ Technology Stack Used

*   **Frontend:** React.js,Next js, Tailwind CSS, DaisyUI, React Router DOM
*   **Backend:** Node.js, Express.js
*   **Database:** MongoDB
*   **Authentication:** Firebase Auth, JWT (JSON Web Token)
*   **State & UI Alerts:** React Hot Toast / React Icons
